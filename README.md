

#  Head of Data Project

##  Overview
This project covers the main steps of a data workflow:
web scraping, data processing, analysis, and visualization.

---

## Session 1 — Introduction & Setup
- Set up the working environment (Python, notebooks, libraries)
- Discovered the structure of a data project
- First manipulations with basic tools

---

## Session 2 — Web Scraping
- Scraped IMDB Top 250 movies using `requests` + `BeautifulSoup`
- Extracted title, year, and rating
- Scraped an e-commerce website using `Selenium`
- Handled dynamic content (scroll, loading)
- Built reusable functions to extract product data

---

## Session 3 — Data Exploration (EDA)
- Loaded datasets with `pandas`
- Cleaned and formatted data
- Explored data using statistics (`describe`, `groupby`)
- Identified patterns and insights

---

## Session 4 — Data Science
- Applied basic data processing techniques
- Prepared data for analysis
- Used Python tools to manipulate datasets

---

## Session 5 — SQL
- Wrote SQL queries to extract and analyze data
- Used joins, filtering, and aggregations
- Worked with structured databases

---

## Session 6 — Dashboard / Visualization
- Built visualizations to present insights
- Used charts to make data understandable
- Focused on clear and simple data storytelling

---

##  Tools Used
- Python (Pandas, BeautifulSoup, Selenium)
- SQL
- Jupyter Notebook
- PowerBI

---

##  Conclusion
This project allowed me to go through the full data pipeline, from data collection to analysis and visualization, using real-world data and tools.
